# JavaScript DOM and Event Handling Task

This is a simple interactive web page built using HTML, CSS, and JavaScript. 

## Project Features
* A main greeting header that says "Hello".
* Four separate boxes labeled red, blue, green, and yellow.
* Clicking any box changes its background color to the respective color.
* An input field to enter the user's name.
* A "Greet" button that updates the header text to welcome the user by name.

## Files Included
* `index.html` - Contains the structure of the webpage.
* `style.css` - Contains the basic styling and layout.
* `script.js` - Contains the DOM manipulation and click event logic.
* `Readme.md` - Documentation of the project.

## How to Run
1. Open the folder containing the files.
2. Double-click the `index.html` file to open it in any web browser.
3. Type your name in the input box and click "Greet".
4. Click on the boxes to see the colors change.
